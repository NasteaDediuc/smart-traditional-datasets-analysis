# Smart Home Energy Consumption Analysis

Acest proiect conține analiza consumului energetic al unei locuințe inteligente pe baza datelor temporale și a consumului pe echipamente. Sunt investigate evoluția consumului zilnic, profilul orar, distribuția consumului total, corelațiile dintre variabile și impactul unor factori externi, precum temperatura.

Analiza include calculul unor indicatori esențiali, cum ar fi:
- consumul total kWh/lună;
- consumul mediu lunar și zilnic;
- consumul total anual;
- consumul minim și maxim zilnic;
- intervalul orar și ora de vârf;
- consumul mediu și maxim la ora de vârf;
- consumul maxim lunar;
- puterea medie observată pentru locuința smart.

Pentru interpretarea rezultatelor sunt utilizate grafice interactive realizate cu Plotly, iar prelucrarea datelor este efectuată în Python, cu ajutorul bibliotecilor PySpark, Pandas și NumPy.
